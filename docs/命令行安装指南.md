# Windows 命令行安装指南

## 🎯 三种安装方式

Windows 11 支持通过命令行安装软件，有三种方式：

1. **winget**（推荐）- Windows 11 自带，无需额外安装
2. **chocolatey** - 第三方包管理器，功能强大
3. **scoop** - 轻量级包管理器

---

## 方式一：winget（推荐，Windows 11 自带）

### 检查是否可用

```powershell
winget --version
```

如果显示版本号，说明已可用。如果没有，需要更新 Windows 11。

### 安装 Node.js

```powershell
# 安装 Node.js LTS 版本
winget install OpenJS.NodeJS.LTS

# 或者安装最新版本
winget install OpenJS.NodeJS
```

### 安装 Git

```powershell
winget install Git.Git
```

### 安装 VS Code（可选）

```powershell
winget install Microsoft.VisualStudioCode
```

### 安装完成后

**重要**：安装完成后需要**重启 PowerShell** 或**重新打开终端**，环境变量才会生效。

验证安装：
```powershell
# 重新打开 PowerShell 后运行
node --version
npm --version
git --version
```

---

## 方式二：Chocolatey（功能强大）

### 1. 安装 Chocolatey

**以管理员身份运行 PowerShell**，然后执行：

```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
```

### 2. 验证安装

```powershell
choco --version
```

### 3. 安装软件

```powershell
# 安装 Node.js LTS
choco install nodejs-lts -y

# 安装 Git
choco install git -y

# 安装 VS Code（可选）
choco install vscode -y
```

### 4. 刷新环境变量

```powershell
refreshenv
```

或者重新打开 PowerShell。

---

## 方式三：Scoop（轻量级）

### 1. 安装 Scoop

在 PowerShell 中运行：

```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex
```

### 2. 安装软件

```powershell
# 安装 Node.js
scoop install nodejs-lts

# 安装 Git
scoop install git

# 安装 VS Code（可选）
scoop install vscode
```

## ⚠️ 重要提示

### 1. 管理员权限

- **winget**：通常不需要管理员权限
- **chocolatey**：需要管理员权限
- **scoop**：不需要管理员权限（但建议使用）

### 2. 环境变量

安装完成后，需要：
- **重新打开 PowerShell**，或
- 运行 `refreshenv`（如果使用 chocolatey）

### 3. 如果安装失败

如果某个工具安装失败，可以：
1. 检查网络连接
2. 以管理员身份运行
3. 手动从官网下载安装
